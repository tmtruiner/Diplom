# Churn Analytics Frontend

React + TypeScript + Vite интерфейс для просмотра:

- KPI и распределения риска;
- клиентов и customer health score;
- rule-based сегментов;
- рекомендаций по удержанию;
- CSV/JSON-выгрузок.

```powershell
npm install
npm run dev
```

По умолчанию frontend обращается к API по адресу
`http://127.0.0.1:8000`.
